Praxis is a flight computer, datalogger and stability control system for model rockets. It features a bmp280 barometer and MPU6050 6-axis accelerometer/gyro.



Debug logs are integrated for checking  where the code is hanging, design is focused on future expandability.



Data is passed through a Kalman filter to clean noise, and PID control is used to actuate fins to keep the rocket vertical.
